# Potenlab Help

Display all available `/pl:*` commands from the Potenlab team toolkit.

## Available Commands

| Command | Description |
|---------|-------------|
| `/pl:help` | Display this help message showing all available commands |
| `/pl:init` | Initialize project with MCPs, rules, and agents in one command |

## Usage

Simply run any command in Claude Code:

```
/pl:help
```
